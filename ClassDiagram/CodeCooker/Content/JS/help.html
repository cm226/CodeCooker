Classes

Use the toolbar on the left to create a class object in the document.
Propertys
Add a property to a class by right clicking on a class and selecting "Add Property"
To delete a property from a class set the return type and name to an empty string, it will then be removed from the class.
Methods
Add a method to a class by right clicking on a class and selecting "Add Methods"
To delete a method from a class set the return type, name and arguments to an empty string, it will then be removed from the class.
Arguments
Method arguments should be comma sepperated in the format: Type name, Type name, ....

Inheritance Arrows

To add an interitance arrow click and drag from a class dropzone to another class drop zone. Inheretance arrows should begin at the deriveing class and end at the base class. (Multiple inheritance is not supported at this time.)

Code Generation

Once you have your model to generate the code using the File menu from the menu banner select GenCode then select the language. Wate for the code to be generated then you will get a message telling you that the code is ready.
