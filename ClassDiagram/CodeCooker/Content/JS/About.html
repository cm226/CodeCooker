Create Some UML

Using the Class diagram interface you can easily create class objects, Inherate from and implement classes and interfaces using click and drag. Let the diagram logic figure out the details like what type of arrow you should be using.

The current version of code cooker will generate UML 2.0 compliant diagrams (With the exception of associations, coming soon)

Save

Code Cooker integrates with Google Drive, so you can save directly to your google dive from the class diagram interface. By utilizing the google dive service you can rest assured that your data is in safe hands.

Code Generation

The focus of code cooker is to provide code generation for your models. Code Cooker currently provides code generators for C++, C#, CoffeeScript and JSON.

We didn't stop there however. Before generating code for the models, the models are what we call 'Completed'. This means that based on what information is in the model we may add a constructor or make a call to the super().

For example: Say class A has a constructor with an int and a float argument. If you create a class B that derives from A and forget to define the constructor in B, we will insert one for you that takes and int and a float argument.

Type Sugestion

One of the cool new features also released in Version 0.5 is the type suggestion system. This system suggests types that are either defined in your diagram or are part of the common set types {Int, Float, Double, String, Boolean, Date, Time, DateTime}. If one of the common types are used the code cooker language generators will automatically convert that type to the representation used in your selected language.
